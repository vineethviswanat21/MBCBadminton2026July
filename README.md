# String Smashers — MBC Badminton Tournament, July 2026

A colorful, responsive GitHub Pages landing page featuring:

- Tournament-format poster with click-to-enlarge view
- Interactive tournament-notes accordion
- Expand All / Collapse All controls
- Official Google Sheets scoring link
- Sticky navigation and back-to-top button
- Mobile-friendly responsive layout

## Repository structure

```text
.
├── index.html
├── README.md
└── assets
    └── tournament-format.jpeg
```

## Publish using GitHub Pages

1. Upload all files to the repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save and wait for the published URL.

The scoring-sheet URL is already configured in `index.html`.
